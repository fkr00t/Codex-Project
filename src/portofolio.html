Portofolio Kami

Beberapa hasil karya yang telah kami selesaikan untuk klien dari berbagai industri.

Landing Page Basic

Landing page adalah halaman web pertama yang dirancang untuk menarik perhatian pengunjung dan mengarahkan mereka untuk melakukan tindakan spesifik, seperti mengisi formulir,menggunakan jasa dan membeli produk yang di tawarkan.

Web Pemesanan Tiket

Web pemesanan tiket adalah platform online yang memungkinkan pengguna untuk membeli tiket acara atau perjalanan dengan fitur pencarian, pemesanan, pembayaran, dan pengiriman e-ticket, sementara admin dapat mengelola acara, pengguna, pembayaran, serta laporan penjualan melalui dashboard dan fitur manajemen lainnya.

Web Absensi Kayawan

Web Absensi Karyawan dengan Sistem Barcode adalah platform yang memungkinkan karyawan untuk melakukan absensi menggunakan barcode unik, sementara admin dapat mengelola data karyawan, laporan absensi, dan pengaturan melalui dashboard yang aman dan efisien.

Web Portofolio Sekolah

Web Portofolio Sekolah adalah situs yang menampilkan informasi tentang program studi, acara atau event yang diselenggarakan sekolah, serta dilengkapi dengan fitur admin untuk mengelola data program studi, event, galeri, dan pengguna secara efisien.
